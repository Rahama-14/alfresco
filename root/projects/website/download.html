Alfresco Requirements
Alfresco Technology Preview Release requires the following (full details in each Readme below):

Software

  • Java Development Kit 5.0
    http://java.sun.com
  • MySQL Database
    http://www.mysql.com
You may also want to use OpenOffice for document transformation.

Recommended Hardware Specification
Pentium 4 processor with 512Mb of RAM and at least 100Mb of Hard Disk space.
