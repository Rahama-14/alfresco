Why Alfresco? | Why Open Source?

  






Alfresco is an open source, open-standards content repository built by the most experienced content management team that includes the co-founder of Documentum. The Alfresco product has a lean, modular component architecture that allows new functionality to be added without any system disruption and is significantly faster than proprietary commercial systems.



Alfresco web interface.


Desktop file access.
J2EE-based professional portal projects that require a content management system at a much lower cost than commercial systems.
Departments that wish to replace shared drives for storage of information to improve searching, facilitate sharing through browser interfaces and increase compliance through content control.
ISVs and application developers who wish to enhance their applications, such as CRM or ERP, with a low-cost, royalty-free, super-fast content management system.
Low-cost, open source alternative to Microsoft SharePoint™.


An enterprise-scale repository providing a web service API, fulltext indexing and retrieval, multiple classification schemes, team collaboration spaces and an extensible aspect framework that includes versioning, locking and content streaming.
Out-of-the-box portal solutions including portlets for repository browsing with multiple views, wizards for administration, content handling, version handling, team collaboration spaces, in-line editing of web content, desktop integration for seamless launching and saving of content.
Lean, modular component architecture plugs-in using Aspect Oriented Programming.
Faster execution paths to searching and databases.
Microsoft-compatible and CIFS-compatible file store access with automatic synchronization.
Automated content processing including auto-classification, auto-indexing and workflow handling.
Content standards support (JSR-168, WebDAV, Microsoft File System-compatible) means no vendor lock-in.
