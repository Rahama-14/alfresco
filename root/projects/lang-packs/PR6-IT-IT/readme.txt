======================
Alfresco Language Pack
======================

For release: PR6

For locale: it_IT (Italiano_Italia) Version 1.0 - 20050929


==============================
Contents of this Language Pack
==============================

action-config_it_IT.properties

messages_it_IT.properties

rule-config_it_IT.properties

============
Installation
============

- Copy 'messages_it_IT.properties' file into <config root> folder.

- Copy 'action-config_it_IT.properties' file into <config root>/messages folder.

- Copy 'rule-config_it_IT.properties' file into <config root>/messages folder.

- Edit the 'web-client-config.xml' file in the <config root> folder to set what languages
  you wish to be available:

  - Find the '<languages>' section
  - Add or remove languages of the form:

       '<language locale="it_IT">LangName</language>'

- Save the file

- Restart the Alfresco server
