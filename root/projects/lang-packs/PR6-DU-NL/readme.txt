======================
Alfresco Language Pack
======================

For release: PR6

For locale: du-NL (Dutch Netherlands)


==============================
Contents of this Language Pack
==============================

action-config_du_NL.properties

messages_du_NL.properties

rule-config_du_NL.properties


============
Installation
============

- Copy 'messages_du_NL.properties' file into <config root> folder.

- Copy 'action-config_du_NL.properties' file into <config root>/messages folder.

- Copy 'rule-config_du_NL.properties' file into <config root>/messages folder.

- Edit the 'web-client-config.xml' file in the <config root> folder to set what languages
  you wish to be available:

  - Find the '<languages>' section
  - Add or remove languages of the form:

       '<language locale="du_NL">Dutch</language>'

- Save the file

- Restart the Alfresco server
