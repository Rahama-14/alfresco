Alfresco Web Content Management Enabler
---------------------------------------

After installin the WCM Enabler and restarting your Alfresco server,
you should now find 2 additional spaces in your Alfresco repository:
- Web Projects in Company Home
- Content Forms in Data Dictionary

If you wish to use the Website Preview feature, you may need to start
the Alfresco Virtualization server if it is not started by the standard
Alfresco start script:
- on Windows run virtual_start.bat
- on Linux run sh virtual_start.sh
