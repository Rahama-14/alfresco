Information for the visually impaired

If you have any trouble using our website with certain web browsers or other software like screen readers, we want to put that right. Our web design team are working to ensure we comply with the Web Accessibility Initiative guidelines set out by the W3C organization.

Customer feedback is important to us. If you discover a part of our website that has been difficult to use please let us know by emailing info@virginmoney.com

Braille, large font and audio material

We can provide information on all our products in most formats. Call us on 08456 10 20 20 for more information. Alternatively, send an email to info@virginmoney.com and let us know your name, address, which product you're interested in and the format you'd like (e.g. braille)

Type Talk

Virgin Money welcome calls through the Type Talk service provided by BT. Call 18001 followed by the number you require, you'll then be connected to an operator offering text phone assistance.

Getting in touch

If you would like to get touch by email our address is info@virginmoney.com and our postal address is Virgin Money, PO Box 9522, Chelmsford, CM99 2AB.
