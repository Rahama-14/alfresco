Best Effort Support

Alfresco are aware that once this information is published to coincide with a release, updates to the underlying software will be made available. Provided that the version you are using is later than the version we publish, we will support you. This holds true up until the point where we can prove that the issue is due to the later version of the operating system, database, middleware or browser. We will then work with you to find the best solution, which in a few cases may be to downgrade.
