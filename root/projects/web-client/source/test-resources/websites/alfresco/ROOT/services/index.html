Alfresco Customer Services Network

Alfresco offers a full range of support offerings that can be tailored to your specific needs. Alfresco sees open source support as more than answering the phone. Alfresco believe a full support service encompasses offering:

  • Stable Certified Product Stacks
  • Scalability Benchmark Certified Stacks
  • A Controlled Release Model
  • Maintenance and Updates
  • Bugs and Patch Support
  • Problem Resolution
  • Compatibility Advice
  • Migration Advice
  • Upgrade Support
  • Performance and Tuning Advice
  • Configuration Advice

Using a customer portal to automate

  • Technical Advice
  • Product Upgrade Notification and Distribution
  • Bug Tracking
  • Case Management

Alfresco Customer Service is delivered to give the confidence and guarantee that only comes from the experts who built the product

Alfresco also believe that support should occur through the whole lifecycle of product usage from:

  • Development through
  • Testing
  • Deployment and Production to
  • Backup and Recovery

Alfresco Development Services Network

This is aimed at the developer and offers for the server framework:

  • API Support – Java, PHP, Ruby, .NET
  • Rule and Action Configuration
  • Rule and Action Customization

And for the client framework:

  • Space Template Management
  • Presentation Template/Dashboard Development
  • Client Framework Customization
  • Skinning

This is not included in the Alfresco Production Services Network.

Alfresco Production Services Network

This is designed to deliver a stable, scalable production system with problem resolution and offers:

  • Stable Certified Product Stacks
  • Scalability Benchmark Certified Stacks
  • A Controlled Release Model
  • Maintenance and Updates
  • Bugs and Patch Support
  • Problem Resolution
  • Compatibility Advice
  • Migration Advice
  • Upgrade Support
  • Performance and Tuning Advice
  • Configuration Advice

This does not include the Alfresco Developer Services Network or Alfresco Backup and Recovery Services Network.

Alfresco Backup and Recovery Services Network

This is designed to proactively manage for backup and recovery and offer emergency help. If offers:

  • Backup Configuration Advice
  • Recovery Advice
  • Best Practice
  • Emergency Help to Advise on the Most Rapid Road to Recovery when Necessary
