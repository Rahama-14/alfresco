Platform Overview

The Alfresco Network is the leading open source alternative for Enterprise Content Management. Across the board, corporations, Governments and other organizations are updating their purchasing models to get the benefit of open source. The major drivers are cost and the use of an open architecture and standards.

The Enterprise Networks offer:

  • Certified Stacks
  • Maintenance, Updates and Patch support
  • Customer Support – Problem Resolution, Compatibility and Migration Advice
  • Customer Portal – Information, Bug Tracking and Case Tracking
  • Performance Tuning Advice
  • Indemnity and Warranty

The Community Newtork is unsupported and has no indemnity or warranty.

Architecture

  • Full Service Oriented Architecture using Spring, Hibernate, Lucene, MyFaces, JSR 168, JSR 170 and JSE5
  • High Availability – Master Slave Distributed Synchronization – Within a Data Centre through 2 phase  commit or remotely between 2 sites through replication
  • High Availability Manager – Failover Manager
  • Clustering  – Fully Distributed Cache
  • Zero Footprint Client
  • Benchmark Certification

Administration

  • Simple Installation
  • Invite User Facility
  • Import/Export
  • Security and User Management with Users, Groups and Roles
  • Document Security in Web Client
  • Upgrade and Migration Administration
  • Full Export/Import of Whole Repository
  • Single Sign-On through NTLM
  • LDAP – Authenticate against LDAP Services and Access LDAP Directory Services
  • Zero-Time Upgrade for Development Extensions

Standards Support

  • Web Services, JSR 168, JSR 170 Level 2, MyFaces, WebDAV, DeltaV, SQL, RSS

Customization
